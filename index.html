Clustering Analysis Report for Iris and Image Datasets

The Clustering Analysis Report for Iris and Image Datasets project involves implementing the K-Means clustering algorithm to categorize data from two distinct datasets: the Iris dataset and a collection of handwritten digit images. The project focuses on centroid initialization, normalization, and dimensionality reduction to enhance clustering accuracy. For the Iris dataset, a conventional K=3 approach is validated through SSE and V-measure scores, reflecting clear cluster delineations. The image dataset, being more complex, required extensive analysis, with SSE plots indicating optimal clustering parameters. Despite challenges with the image data, the project demonstrates the effectiveness of K-Means and provides insights into future improvements, including alternate clustering techniques and advanced preprocessing methods. The report includes comprehensive evaluations, highlighting the robustness and precision of the clustering solution.

Machine Learning Classifier Ensemble

The Machine Learning Classifier Ensemble project aims to enhance prediction accuracy by combining multiple classifiers. This comprehensive machine learning initiative involves several key steps: data preprocessing, which includes standard scaling and over-sampling with SMOTE to tackle class imbalance; feature selection using variance threshold and SelectKBest methods; and extensive hyperparameter tuning for various classifiers such as K-Nearest Neighbors, Decision Trees, Gaussian Naive Bayes, and Neural Networks. The core of the project is the implementation of an ensemble model using a voting classifier, which leverages the strengths of individual models for superior predictive performance. Developed in Python and utilizing essential libraries like NumPy, pandas, scikit-learn, and imbalanced-learn, the project also features visualization of cross-validation results and saves final predictions for further analysis. This approach ensures a robust framework for handling complex predictive modeling tasks.

Buffer Manager System

The Buffer Manager System project involves developing the NARU Buffer Management System using C. This system maintains multiple word buffers, similar to open files, allowing users to modify their contents through a shell interface. Key features include creating, closing, reading, writing, and executing commands on buffers. The system supports running processes in both the foreground and background, managing multiple open buffers, and using pipes for inter-process communication. Additionally, it handles process signals for suspending, resuming, and terminating processes, emphasizing Unix-like process management and command execution. The project requires implementing functionality for buffer management, process control, and signal handling, ensuring efficient and organized buffer operations.

The TinySF Software Floating-Point Library project entails creating a custom 12-bit floating-point library for the MUAN Programming Language, tailored for embedded systems without hardware floating-point support. PixelPioneer: 2D Linked List Image Manipulation

The PixelPioneer project involves developing a two-dimensional doubly-linked list to represent and manipulate grayscale images.
